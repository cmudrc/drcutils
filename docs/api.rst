API
===

This page documents the supported top-level public API from
``drcutils.__all__``.

Top-level groups:

- Module facades: ``brand``, ``data``, ``runtime``, ``visualization``, ``viz`` (legacy alias)
- Brand utilities: ``COLORS``, ``flag``, ``watermark``
- Data utility: ``convert``
- Runtime helpers: ``is_notebook``, ``is_google_colab``
- Visualization helpers: ``export_figure``, ``get_figure_preset``,
  ``visualize_network``, ``visualize_stl``
